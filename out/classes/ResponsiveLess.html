getLessVars parses your LESS variables to Javascript (provided you make a dummy node in LESS)

Parameters:

  • id String

    The CSS-id your variables are listed under.

  • [parseNumbers=true] Boolean optional

    Try to parse units as numbers.

Returns:

Object:

A value object containing your LESS variables.

Example:

LESS: @myLessVariable: 123px;

#dummyLessId { width: @myLessVariable; }

Javascript: getLessVars('dummyLessId'); returns: {myLessVariable:123}
